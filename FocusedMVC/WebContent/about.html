About Focused

Focused is a collaborative Skill Distillery boot-camp project. This project was completed in 7 days during week 10 of our Full-stack Java Web-app Development Program by Elijah Ginter, Patrick Gaston and Steve Nagle.

Back-end Technologies used in our application include: Java Persistence API, MySQL Database, Spring MVC, JPA, Hibernate, running on an Apache Tomcat server and using Maven dependencies including c3p0

Front-end Technologies used in our application include: HTML5, CSS3, Twitter Bootstrap templates and JavaScript
